## Quick start

```sh
git clone https://github.com/gregruthenbeck/cavapa_docs.git
cd mkdocs-material-boilerplate
pipenv sync --dev
pipenv shell
inv serve --config-file cavapa_docs.yml -d localhost:8008
```
